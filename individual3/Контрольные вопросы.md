# Контрольные вопросы

1. **Каким образом можно получить доступ к элементу на веб-странице с помощью JavaScript?**
   - Для доступа к элементам можно использовать методы `document.getElementById()`, `document.querySelector()`, `document.querySelectorAll()`, и другие DOM методы.

2. **Что такое делегирование событий и как оно используется для эффективного управления событиями на элементах DOM?**
   - Делегирование событий — это техника, при которой обработчик события устанавливается на родительский элемент, а события от потомков перехватываются благодаря всплытию. Это позволяет эффективно управлять событиями на множестве элементов.

3. **Как можно изменить содержимое элемента DOM с помощью JavaScript после его выборки?**
   - Содержимое элемента можно изменить, используя свойства `innerHTML`, `textContent`, или методы, такие как `appendChild()`.

4. **Как можно добавить новый элемент в DOM дерево с помощью JavaScript?**
   - Новый элемент можно создать с помощью `document.createElement()` и добавить его в DOM дерево, используя методы `appendChild()` или `insertBefore()`.
